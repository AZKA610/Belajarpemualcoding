Tutorial HTML Pemula

1. Buka Text Editor
Gunakan aplikasi seperti Notepad atau Visual Studio Code.
2. Buat File Baru
Klik File → New untuk membuat file baru.
3. Tulis Kode HTML Dasar
Ketik kode berikut: 
<!DOCTYPE html>
<html>
<head>
    <title>Halaman Pertama</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
</body>
</html>
4. Simpan File
untuk awal simpan dulu dengan namaindex.html
5. Buka di Browser
Klik dua kali file HTML atau buka di Google Chrome atau firefox jika kalian menambhakan codingan baru saat sudah di save terus masuk ke browser dan reload page.
