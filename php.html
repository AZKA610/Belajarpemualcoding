Tutorial PHP Pemula

1. Install Software
Download dan install XAMPP dan setelah di download nyalakan 2 module Apache dan MySql dan setelah itu pencet tombol admin di samping start di MySQL.
2. Buka Folder htdocs
Masuk ke folder htdocs di dalam file directory XAMPP.
3. Buat File PHP
Buat folder baru terlebih dahulu dengan nama belajar dan masuk ke dalam folder yang baru dibbuat daqn bikin file dengan nama index.php
4. Tulis Kode PHP Dasar
Contoh kode: 
<?php
echo "Halo Dunia dari PHP!";
?>
5. Jalankan di Browser
Buka browser dan ketik:
http://localhost/index.php
